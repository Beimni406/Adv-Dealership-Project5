# Car Dealership App

## Overview
This Java console app lets users view, search, add, and remove vehicles from a dealership inventory.  
It also manages sales and lease contracts and saves all data to CSV files.

## Features
- Search vehicles by price, make/model, year, color, mileage, and type
- Add and remove vehicles
- Calculate total and monthly payments for sales and leases
- Save dealership and contract data to CSV files

## How to Run
1. Open the project in IntelliJ IDEA.
2. Make sure `dealership.csv` is in the project folder.
3. Run the program from `CarDealership.java`.


- Java (OOP)
- IntelliJ IDEA
